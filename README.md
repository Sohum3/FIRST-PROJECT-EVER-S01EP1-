# FIRST-PROJECT-EVER-S01EP1-
🏨 Hotel Management System (My First Project)

This is my first ever programming project, built using Python and Tkinter.
It is a simple GUI-based application that simulates basic hotel front desk operations like check-in and check-out.

🚀 Features
🧾 Guest Check-In
Enter name, address, phone number
Select room type
Choose payment mode
🚪 Guest Check-Out
Enter room number
Simulates checkout process
📋 Show Guest List
Placeholder for displaying all guests
🔍 Fetch Guest Info
Placeholder for retrieving guest details
❌ Exit Application
Close the program
🖥️ Tech Stack
Language: Python 🐍
GUI Library: Tkinter
▶️ How to Run
1. Clone the repository
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
2. Run the application
python hotel_management.py
⚠️ Requirements
Python 3.x
Tkinter (comes pre-installed with most Python distributions)

If Tkinter is missing:

sudo apt-get install python3-tk   # Linux
